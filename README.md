# Snapchat-Registration-Checker
![Banner](./image.png)
Advanced async tool to check usernames, emails, and phone numbers using internal endpoints.

Built for speed, reliability, and large-scale checking with proxy rotation and retry logic.

--------------------------------------------------

🚀 Features

- ✅ Check:
  - Usernames
  - Emails
  - Phone numbers

- ⚡ High-performance (async + threading)
- 🔁 Smart retry system (up to 3 attempts)
- 🌐 Proxy support with rotation + cooldown
- 🧠 Auto-detect input type (user / email / phone)
- 📂 Auto-save results to output files
